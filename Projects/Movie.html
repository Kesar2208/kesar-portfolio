Movie Recommendation System

Project Overview

This project is a content-based movie recommendation system developed using Python. It suggests movies based on genre similarity using TF-IDF vectorization and cosine similarity techniques, widely used in natural language processing (NLP).

Tools Used: Python, Pandas, Scikit-learn
Algorithm: TF-IDF + Cosine Similarity
Dataset: Custom movie dataset (finalfile.csv)

View Dataset

Objective

To help users discover movies similar to the ones they already enjoy — saving time and improving discovery. This was achieved by comparing genre metadata across all movies.

How the System Works

  1. Data Loading: Loads a dataset (finalfile.csv) with movie titles and genres.
  2. Text Vectorization: Applies TF-IDF to convert genres into numeric vectors.
  3. Cosine Similarity: Calculates similarity scores between movies.
  4. Recommendation: Based on a user-selected title, it returns 10 similar movies.

Core Code Snippet

def get_recommendations(Movies, cosine_sim=cosine_sim):
    idx = movies.loc[movies['Movies'] == Movies].index[0]
    sim_scores = list(enumerate(cosine_sim[idx]))
    sim_scores = sorted(sim_scores, key=lambda x: x[1], reverse=True)
    sim_scores = sim_scores[1:11]
    movie_indices = [i[0] for i in sim_scores]
    return movies['Movies'].iloc[movie_indices]
