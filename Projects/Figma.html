Gym & Fitness Website

Project Overview

This project involved designing a visually clean and energetic website for a fictional gym and fitness brand using Figma. The focus was on showcasing fitness offerings with clarity, strong branding, and easy navigation across key user-centric pages.

Tool Used: Figma
Role: UI/UX Designer
Design Type: Desktop Web Design

Goal

To create a compelling gym website that:

  • Highlights the gym’s services and benefits
  • Provides a seamless experience for new users to understand offerings
  • Encourages actions like sign-ups, inquiries, and membership purchases

Key Pages Designed

Home Page
• Bold hero section with motivational tagline and “Get Started Now” CTA
• Introduction to gym services, features, and testimonials
• Section previewing “What You Get Here”

About Us
• Details about the gym’s mission, team, and values
• Showcases credibility and builds trust with visitors

Services (What You Get)
• Clean card-based layout presenting all gym facilities and offers
• Includes icons, short titles, and brief descriptions

Pricing Page
• Simple pricing table layout
• Clear breakdown of Basic & Premium plans
• Includes CTAs to sign up or contact for custom plans

Contact Us Page
• Contact form with fields for name, email, message
• Google Maps-style embed or location display
• Footer with essential links and contact info

Design Highlights

  • Consistent color palette: Bold black, white, and red
  • Typographic hierarchy that promotes readability and impact
  • Layout built using a grid for clean alignment and responsiveness
  • Interactive CTA buttons for easy user engagement

Outcomes & Learnings

  • Improved confidence in full-page UI layout creation
  • Learned to maintain branding consistency across multiple sections
  • Focused on optimizing for clarity and user action
